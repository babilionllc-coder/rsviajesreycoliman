## AEO Optimization Checklist

When creating or editing content, follow this checklist:

### Content Structure for AEO
- [ ] Use the **question as the exact H2/H3 heading**
- [ ] Provide a **direct 20-60 word answer** immediately after the heading
- [ ] Then expand with supporting details, lists, and examples
- [ ] Use **natural conversational tone** (as if speaking to a friend)
- [ ] Include **location context** (Colima, México) in answers

### Schema Markup for AEO
- [ ] Use `FAQPage` schema for all FAQ sections
- [ ] Use `Speakable` schema for key answers (already implemented)
- [ ] Use `TouristTrip` schema on each trip detail page
- [ ] Use `TravelAgency` schema on homepage
- [ ] Use `BreadcrumbList` schema on all subpages
- [ ] Include `areaServed` in TravelAgency schema

### Meta Tag Formula
```
Title: [Primary Keyword] — [Secondary Keyword] | RS Viajes Rey Colimán
Description: [Action verb] [primary keyword]. [Value prop] [location]. [150-160 chars total]
```

**Example:**
```
Title: Viajes Internacionales — Salidas Garantizadas | RS Viajes Rey Colimán
Description: Explora nuestras salidas garantizadas a Europa, Asia y más. Paquetes todo incluido con vuelos y hospedaje. RS Viajes Rey Colimán, Colima, México.
```
